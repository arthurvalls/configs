One command

From inside the folder, on her Debian/Ubuntu machine:

$ ./install.sh

Safe to run again — it skips whatever's already there and backs up anything it would replace. Add --dry-run to see what it would do first.

Making it hers

Small, friendly knobs — no config rabbit-holes.

change the coloursedit ~/.config/kitty/totoro.conf — it updates the instant she saves
bigger / smaller textCtrl + and Ctrl - (Ctrl 0 resets)
tweak the promptedit ~/.config/starship.toml
add a shortcutedit ~/.config/fish/config.fish
